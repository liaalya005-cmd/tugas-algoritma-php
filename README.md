# Judul Program

## Program Menghitung Rata-Rata Nilai Mahasiswa Menggunakan PHP


# Deskripsi

Program Menghitung Rata-Rata Nilai Mahasiswa merupakan program sederhana berbasis PHP yang digunakan untuk menghitung rata-rata nilai dari beberapa mata kuliah yang diinput oleh pengguna. Program ini menerapkan konsep dasar algoritma dan pemrograman, seperti variabel, input dan output, operator, serta percabangan untuk menghasilkan perhitungan yang akurat dan mudah dipahami.


# Tujuan Program

- Memahami dasar-dasar algoritma dan pemrograman.
- Menerapkan konsep PHP dalam menyelesaikan perhitungan sederhana.
- Mengolah data nilai mahasiswa menjadi informasi yang mudah dipahami.
- Melatih kemampuan membuat program yang terstruktur dan mudah dikembangkan.


# Materi yang Digunakan

Program ini menerapkan beberapa materi dasar pemrograman, yaitu:

- Variabel
- Input dan Output
- Operator Aritmatika
- Percabangan (if-else)
- Fungsi Sederhana

  
# Fitur Program

- Memasukkan nama mahasiswa.
- Memasukkan nilai beberapa mata kuliah.
- Menghitung total seluruh nilai.
- Menghitung nilai rata-rata secara otomatis.
- Menampilkan hasil perhitungan dengan jelas.
- Menampilkan keterangan berdasarkan nilai rata-rata.

# Cara Menjalankan Program

1. Clone atau download repository GitHub.
2. Jalankan aplikasi XAMPP atau Laragon.
3. Aktifkan Apache.
4. Salin folder **tugas-algoritma-php** ke dalam folder **htdocs**.
5. Buka browser.
6. Akses alamat berikut:

```
http://localhost/tugas-algoritma-php
```

7. Masukkan data mahasiswa dan nilai mata kuliah.
8. Klik tombol proses.
9. Program akan menampilkan total nilai, nilai rata-rata, dan keterangan.


# Contoh Output

**Input**

```
Nama Mahasiswa : Andi
Nilai Algoritma : 80
Nilai Basis Data : 85
Nilai Pemrograman Web : 90
```

**Output**

```

Rata-rata : 85
Keterangan : Lulus
```
